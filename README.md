# 🤖 AI-Powered HR Recruitment Automation

## Overview

An end-to-end AI recruitment automation workflow built using n8n, OpenAI, Google Drive, Google Sheets, Google Docs, and Gmail.

This system automates the hiring pipeline from resume submission to candidate shortlisting and technical assessment generation.

---

## Features

✅ Resume Upload & Storage

✅ Resume Parsing & Information Extraction

✅ Candidate Profile Generation

✅ AI-Based JD Matching

✅ Candidate Scoring & Ranking

✅ Automated Shortlisting

✅ ATS Tracking in Google Sheets

✅ Technical Assessment Generation

✅ Automated Email Notifications

---

## Workflow Architecture

1. Candidate submits application form
2. Resume uploaded to Google Drive
3. Resume content extracted
4. AI generates candidate profile
5. Candidate matched against Job Description
6. AI calculates suitability score
7. Candidate classified:
   - Shortlisted
   - Maybe
   - Rejected
8. Results stored in ATS Sheet
9. Technical assessment generated
10. Email sent to shortlisted candidates
